# Kyoto Night Shrine

Small Unity 2D MVP prototype for a Kyoto shrine/shop management and nighttime purification game.

The current repository is an intentionally light starter project. It includes the first folder structure, three placeholder scenes, and planning documentation, but no complex gameplay systems yet.

## Starter Scenes

- `Assets/Scenes/ShopShrine.unity`
- `Assets/Scenes/NightMap.unity`
- `Assets/Scenes/Result.unity`
